# emacs_setting
## Included features
- lint
  - llvm-mode
  - yaml-mode
- neo-tree
  - f8
- xcscope
- redo
- hide menu bar
- window dedication
- reverse window
- marmalate and melpa package link
- magit(manually install)
- VisualStudio-like debugging key setting
  - f5 : continue
  - f10 : next
  - f11 : step
